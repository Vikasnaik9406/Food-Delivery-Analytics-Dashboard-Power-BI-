# Food-Delivery-Analytics-Dashboard-Power-BI-
ZomGo is an end-to-end Power BI analytics project built to analyze a food delivery business. The dashboard provides actionable insights into orders, quantities, customer behavior, delivery status, and payment methods, enabling management to make data-driven decisions.
